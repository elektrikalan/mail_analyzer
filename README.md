# 📧 Mail Analyzer Pro

**Mail Analyzer Pro**, Microsoft Outlook hesapları ve çevrimdışı PST dosyaları üzerinde derinlemesine güvenlik analizi ve arama yapabilen, modern arayüzlü bir masaüstü uygulamasıdır.

---

## 🇹🇷 Türkçe Tanıtım

### 🚀 Özellikler
- **Gelişmiş PST Tarayıcı:** Tüm Outlook hesaplarını ve bağlı PST dosyalarını rekürsif (iç içe) olarak tarar.
- **Akıllı Risk Analizi:** E-posta içeriklerini anahtar kelime ve Regex (IP, Email, JWT vb.) kurallarıyla analiz ederek risk skoru atar.
- **Global Arama:** Tüm klasörler ve PST'ler içinde konu veya içerik bazlı gelişmiş arama yapabilir.
- **Profesyonel Dashboard:** Güvenlik bulgularını ve istatistikleri modern bir panel üzerinden sunar.
- **Excel Raporlama:** Analiz sonuçlarını tek tıkla Excel (.xlsx) formatında dışa aktarır.
- **Modern Arayüz:** CustomTkinter ile tasarlanmış, kullanıcı dostu Karanlık Mod (Dark Mode) desteği.

### 🛠️ Kurulum
1. Python (3.10+) yüklü olduğundan emin olun.
2. Klasik Microsoft Outlook uygulamasının bilgisayarınızda yüklü olması gerekir.
3. Gerekli kütüphaneleri yükleyin:
   ```bash
   pip install -r requirements.txt
   ```
4. Uygulamayı başlatın:
   ```bash
   python gui_app.py
   ```

### 📖 Kullanım
- **Dashboard:** Genel güvenlik durumunu takip edin.
- **PST Explorer:** Klasör ağacından istediğiniz klasörü seçip "Scan" butonuna basarak analizi başlatın.
- **Global Search:** Belirli kriterlere göre tüm Outlook'ta arama yapın.
- **Analysis Results:** Tespit edilen tehditleri inceleyin ve rapor alın.

---

## 🇺🇸 English Description

### 🚀 Features
- **Advanced PST Explorer:** Recursively scans all Outlook accounts and connected PST/Archive files.
- **Smart Risk Analysis:** Analyzes email content using keyword and Regex (IP, Email, JWT, etc.) patterns to assign risk scores.
- **Global Search:** Performs advanced searches across all folders and PSTs based on subjects or content.
- **Professional Dashboard:** Displays security findings and statistics through a high-end dashboard.
- **Excel Reporting:** Exports analysis results to Excel (.xlsx) format with a single click.
- **Pro UI:** A modern, user-friendly Dark Mode interface built with CustomTkinter.

### 🛠️ Installation
1. Ensure Python (3.10+) is installed.
2. Classic Microsoft Outlook desktop application must be installed on your system.
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python gui_app.py
   ```

### 📖 Usage
- **Dashboard:** Monitor the overall security state.
- **PST Explorer:** Navigate the folder tree, select a folder, and click "Scan" to start analysis.
- **Global Search:** Search through all accounts using specific filters.
- **Analysis Results:** Review detected threats and export reports.
